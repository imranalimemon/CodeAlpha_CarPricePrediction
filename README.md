# Car Price Prediction 🚗

## CodeAlpha Data Science Internship — Task 3

### Overview
Predict car prices based on features like **brand**, **engine size**, **horsepower**, **mileage**, **fuel type**, and **transmission**. Multiple regression models are compared to find the best predictor.

### Models Used
- Linear Regression
- Ridge Regression
- Lasso Regression
- Random Forest Regressor
- Gradient Boosting Regressor

### Features
- Comprehensive EDA with price distribution, brand analysis, and scatter plots
- Feature engineering and label encoding for categorical variables
- 5-Fold Cross Validation
- Actual vs Predicted scatter plot
- Feature importance analysis

### Tech Stack
- Python, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn

### How to Run
```bash
pip install -r requirements.txt
python car_price_prediction.py
```
